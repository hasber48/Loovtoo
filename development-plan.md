## Этап 1: Счетчик правильных ответов
**Цель:** научиться работать с переменными для хранения состояния

### Что нужно сделать:
1. Создать переменную для хранения счета в начале файла
2. При правильном ответе увеличивать счет на 1
3. Обновлять текст в элементе `<span id="score">` с помощью `.textContent`
4. При неправильном ответе ничего не менять

### Новые концепции:
- Переменные для хранения числовых данных
- Операция `+= 1` для увеличения значения
- Обновление содержимого элемента через `.textContent`

## Важные принципы обучения

### 1. Один шаг за раз
Не переходить к следующему этапу, пока текущий не работает полностью

### 2. Понимание важнее копирования
Лучше написать 5 строк кода самостоятельно, чем скопировать 50

### 3. Эксперименты приветствуются
Изменяй значения, смотри что происходит, ломай и чини код

### 4. Тестирование каждого изменения
После каждого изменения открывай страницу в браузере и проверяй

### 5. Комментарии в коде
Пиши комментарии к непонятным местам своими словами
