# Geo Quiz - Playwright Tests

Автоматизированный тест Playwright, который проходит полностью один из разделов квиза и проверяет все функции.

## Установка

1. Установите зависимости:
```bash
npm install
```

## Запуск тестов

### Базовый запуск (headless режим)
```bash
npm test
```

### Запуск с визуальным отображением браузера
```bash
npm run test:headed
```

### Запуск в debug режиме
```bash
npm run test:debug
```

### Просмотр отчета
```bash
npm run test:report
```

## Что проверяет тест

Тест `test.spec.js` полностью проходит секцию "Europe - Capitals" и проверяет:

✓ **Start Screen**
  - Видимость стартового экрана
  - Наличие заголовка "Geo Quiz"
  - Работа кнопки Start

✓ **Game Selection Screen**
  - Появление экрана выбора
  - Видимость всех 8 вариантов квизов
  - Выбор раздела Europe - Capitals

✓ **Game Screen**
  - Загрузка экрана игры
  - Инициализация счета (0)
  - Загрузка первого вопроса
  - Наличие 4 вариантов ответа
  - Скрытие кнопки Next Question в начале

✓ **Game Flow**
  - Видимость Next Question при выборе ответа
  - Обновление счета после ответа
  - Загрузка следующего вопроса
  - Скрытие Next Question перед новым вопросом
  - Циклический проход через все вопросы

✓ **Game Over Screen**
  - Появление экрана Game Over
  - Отображение финального счета
  - Работа кнопки Restart
  - Возврат к стартовому экрану

## Требования

- Node.js (v14+)
- Python (для локального HTTP сервера, если используется)
- Современный браузер (Chrome/Chromium)
